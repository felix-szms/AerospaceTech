## 核心知识

### 一、

<ScienceDiagram type="quadcopter" title="🚁 四旋翼无人机原理" caption="对角线同向旋转 + 相邻反向旋转，抵消反扭矩" />
四旋翼飞行原理：四只"风扇"如何让无人机飞起来

四旋翼无人机（Quadcopter）没有像直升机那样的"倾斜主旋翼 + 尾桨"结构，它**全靠调节四个旋翼的转速**来实现所有飞行动作。这听起来不可思议，但背后是精妙的力学设计。

#### 1. 升力的产生

每个旋翼高速旋转时，把空气向下推，空气的反作用力就给旋翼一个**向上的升力**。四个旋翼同时加速，总升力超过重力，无人机就升空。

#### 2. 反扭矩与旋翼转向配置

这里有个关键问题：旋翼转动时，空气会给电机一个**反向力矩（反扭矩）**。如果四个旋翼同向旋转，无人机会不停自转。解决方案是——

> **对角线上的两个旋翼同向旋转，相邻的两个旋翼反向旋转。**

即 1、3 号旋翼顺时针（CW），2、4 号旋翼逆时针（CCW）。这样四个反扭矩两两抵消，无人机在正常状态下不会自转。

| 旋翼位置 | 转向 | 螺旋桨类型 |
|---|---|---|
| 前左（1 号）| 顺时针 CW | 正桨（标记 R）|
| 前右（2 号）| 逆时针 CCW | 反桨（标记 L）|
| 后左（3 号）| 逆时针 CCW | 反桨（标记 L）|
| 后右（4 号）| 顺时针 CW | 正桨（标记 R）|

> 💡 **装机常识**：装错螺旋桨方向，无人机一起飞就会疯狂自转甚至翻机。这也是为什么换桨时一定要核对 CW/CCW 标记。

#### 3. 差速控制：四种基本运动

无人机没有舵面、没有副翼，**所有姿态变化都通过改变四个电机的相对转速实现**，这就叫"差速控制"。记住下面四条口诀：

| 运动 | 操控方式 | 物理效果 |
|---|---|---|
| **升降（油门 Throttle）** | 四个桨**同时加/减速** | 总升力增减 → 整体升/降 |
| **俯仰 Pitch（前后）** | **前桨减速、后桨加速** | 机头前倾 → 向前飞 |
| **横滚 Roll（左右）** | **左桨减速、右桨加速** | 机身侧倾 → 向左/右飞 |
| **偏航 Yaw（自转）** | **同向旋翼加速、反向旋翼减速** | 反扭矩失衡 → 绕竖直轴自转 |

> 🎤 **关键理解**：偏航是最反直觉的一条。它利用的正是"对角同向"配置——当你让两个顺时针旋翼加速、两个逆时针旋翼减速时，**升力总和不变（不升不降）**，但**反扭矩不再平衡**，机身就绕竖直轴转动。妙就妙在：产生偏航的力和产生升力的力被巧妙解耦了。

### 二、无人机硬件系统：六大部件

把一架无人机拆开，你会看到下面这些"器官"，每一个都不可或缺：

| 部件 | 比喻 | 功能 |
|---|---|---|
| **飞控（Flight Controller）** | 大脑 | 处理传感器数据，每秒数百次计算并发出电机指令，维持姿态稳定 |
| **电调（ESC，电子调速器）** | 神经 | 将飞控信号转为电机转速控制，每个电机配一个电调 |
| **无刷电机（Motor）** | 肌肉 | 高效、长寿、响应快，转速可达上万 RPM |
| **螺旋桨（Propeller）** | 手脚 | 产生升力，常见尺寸 5–10 英寸，分正反桨 |
| **电池（LiPo 锂聚合物）** | 心脏 | 高放电倍率，电压 3.7V/节（1S），常见 3S–4S |
| **机架（Frame）** | 骨骼 | 承载所有部件，碳纤维/塑料材质，要求轻且刚 |

> ⚠️ **锂电池安全**：LiPo 电池过充、过放、短路或物理破损可能**起火爆炸**。充电必须用平衡充电器，存放电压保持 3.8V/节，鼓包电池立即停用。

信号流向：**遥控器 → 接收机 → 飞控 → 电调 → 电机 → 螺旋桨**。理解这条链路，就理解了无人机的"神经系统"。

### 三、传感器：无人机的"五官"

没有传感器，无人机只是瞎飞的"四个风扇"。飞控依靠下面这些传感器实时感知自身状态：

| 传感器 | 感知什么 | 用途 |
|---|---|---|
| **IMU（惯性测量单元）** | 陀螺仪测角速度、加速度计测线加速度 | **核心**：感知姿态（俯仰/横滚/偏航角），毫秒级响应 |
| **气压计** | 大气压力 | 推算**高度**（气压随高度递减），用于定高 |
| **GPS** | 经纬度、速度 | 户外**定位**、自动返航、航线飞行 |
| **磁力计（电子罗盘）** | 地磁场方向 | 确定**航向**，校准偏航角，受金属干扰需校准 |
| **光流传感器** | 地面纹理移动 | **室内/无 GPS 环境**定位（类似光学鼠标）|

> 💡 **为什么需要这么多传感器？** 没有任何一个传感器是完美的：IMU 会漂移，气压计受气流影响，GPS 在室内失灵，磁力计遇铁磁物失真。飞控通过**传感器融合**（如卡尔曼滤波）把它们的长处组合、短处互补，才能得到稳定可靠的姿态与位置估计。这也是现代航天器导航的共通思路。

### 四、无人机分类与法规

无人机不是想飞就能飞。2024 年 1 月 1 日起，中国**《无人驾驶航空器飞行管理暂行条例》**正式施行，这是我国首部专门规范无人驾驶航空器的行政法规。

#### 1. 按重量分类（关键考点）

| 类别 | 起飞重量 | 典型机型 | 飞行限制 |
|---|---|---|---|
| **微型** | < 0.25 kg | 玩具级、小型航拍 | 飞行高度 ≤ 50 m，视距内 |
| **轻型** | < 4 kg（非微型）| 大疆 Mini/Air 系列 | 视距内，需避让有人航空器 |
| **小型** | < 25 kg（非轻/微）| 大疆 Mavic、行业机 | 需取得相应资质 |
| 中型/大型 | ≥ 25 kg | 农业植保、专业影视 | 严格资质与审批 |

#### 2. 三条硬性规定

- **实名登记**：除微型外的无人机所有者须在**民用无人驾驶航空器综合管理平台（UOM）**实名登记，张贴登记号。
- **禁飞区**：机场净空区、军事禁区、核设施、重要政务区等划设禁飞/限飞区，违规飞行涉嫌违法。
- **操控资质**：小型及以上需取得**操控员执照**；轻型在部分空域也需培训。

> 🏆 **赛事对接**：全国青少年航天创新大赛的"无人机编程技能挑战赛"要求选手在合规前提下完成编程飞行任务。**理解法规不仅是安全要求，也是比赛常识**。你方案中如果涉及无人机应用，必须考虑法规可行性——否则创意再好也"飞不起来"。

### 五、查新方法：让你的选题"站得住"

> 🏆 本课开始的查新训练，直接关系到期末作品包第 5 件——**查新报告**。这是赛事和科创赛的**必备材料**，证明你的创意"有新意"而非重复别人。

#### 1. 为什么要查新

- **避免重复造轮子**：你以为很新的点子，可能别人五年前就做了。
- **找到创新点**：通过对比已有方案，明确"我新在哪里"。
- **积累参考文献**：研究报告需要引用文献支撑你的科学原理。

#### 2. 查新关键词构建法

把你的一句话选题拆成**"对象 + 技术 + 应用"**三组关键词，用 AND/OR 组合检索。

**示例**（假设选题是"用无人机监测海洋塑料垃圾"）：

- 对象词：无人机 / UAV / 无人机遥感
- 技术词：图像识别 / 机器视觉 / 多光谱
- 应用词：海洋垃圾 / 塑料污染 / 海漂物

检索式：`(无人机 OR UAV) AND (图像识别 OR 机器视觉) AND (海洋垃圾 OR 塑料)`

#### 3. 常用检索源

| 数据库 | 特点 | 适用 |

## 本课小结

- **四旋翼原理**：对角同向、相邻反向抵消反扭矩；升降、俯仰、横滚、偏航全靠**差速控制**。
- **六大部件**：飞控（脑）→ 电调（神经）→ 电机（肌）→ 螺旋桨（手脚）→ 电池（心）→ 机架（骨）。
- **传感器**：IMU 测姿态、气压计测高、GPS 定位、磁力计定向、光流补室内——多传感器融合是关键。
- **法规**：2024 年《暂行条例》施行，微型/轻型/小型分级管理，实名登记、禁飞区、操控资质缺一不可。
- **PBL 里程碑**：本课正式提交《选题确认书》并启动查新——**这是你的项目从"想法"变成"立项"的一刻**。

## 活动一：模拟飞行基础操控（15 分钟）

### 飞行前准备

1. 打开模拟飞行软件，选择**四旋翼（Quadcopter）**机型。
2. 选择**无风、开阔场地**场景（如草地、操场）。
3. 确认遥控器通道映射：
   - **左摇杆**：上下 = 油门（Throttle，升降）；左右 = 偏航（Yaw，自转）
   - **右摇杆**：上下 = 俯仰（Pitch，前后）；左右 = 横滚（Roll，左右）

> 💡 这是"美国手（Mode 2）"布局，全球最通用，大疆等商用机也采用此布局。

### 五项基础操控练习

按下表逐项练习，每项先由教师示范，再由学生操作。每完成一项打 ✓。

| 序号 | 任务 | 操作要点 | 完成标准 | ✓ |
|---|---|---|---|---|
| **1** | **起飞** | 缓慢推油门至 50% 左右，无人机离地 1–2 米 | 稳定离地不侧倾 | |
| **2** | **悬停** | 微调油门保持高度，修正俯仰/横滚保持位置 | 在 2 米高度停留 **≥10 秒**，漂移 < 1 米 | |
| **3** | **平移** | 悬停基础上轻推俯仰/横滚，控制方向移动 | 向前 5 米 → 停 → 退回原位 | |
| **4** | **定向飞行** | 先偏航转 90°，再沿新方向平移前进 | 完成一个"正方形航线"（前进→转向×4）| |
| **5** | **降落** | 移至起降点正上方，缓慢减油门，触地即停 | 轻柔着陆，无反弹、无翻机 | |

> 💡 **反思提问**：在模拟器里，你最难控制的是哪一项？为什么？（多数学生答"悬停"——因为它要求四个通道同时微调，这正是飞控在真实无人机上每秒帮你做几百次的事。）

### 挑战任务（💡 如时间允许）

完成基础五项后，尝试：

- **穿越窄门**：从两个障碍物之间穿过（考验精确定位）。

✅ **完成后**：每组推选操控最好的一位，在全班面前演示"正方形航线"。

## 重难点与突破策略

### 重难点 1：差速控制的物理直觉（尤其偏航）

- **学生困难**：俯仰、横滚尚可理解（谁加速谁翘起），但**偏航**靠"反扭矩失衡"很反直觉。
- **突破**：
  - 用**双手演示**——左右手在身前画相反方向的圈（模拟对角同向、相邻反向），然后"让一组手加速"——身体会自然转动。
  - 强调"升降 = 总和变化；俯仰/横滚/偏航 = 分布变化"，这三种姿态运动**总升力不变**，是关键区分。
  - 配合 B 站"四旋翼差速控制动画"（见素材清单）。

### 重难点 2：把模糊想法变成可立项的选题 ⭐【本课最关键】

- **学生困难**：常见三种毛病——
  - **太空泛**："我要做无人机监测环境"（没有对象、场景、方法）
  - **太狭窄**："我要做一个红色的小无人机"（没有问题、没有创新）
  - **超能力**："我要做无人机送货到月球"（脱离科学可行性）
- **突破**：用**"三问法"**引导每组当堂打磨选题：
  1. **你解决谁的问题？**（明确用户/场景）
  2. **你用什么科学/技术？**（明确原理/方法）
  3. **你和现有方案有什么不同？**（明确创新点，靠查新回答）
- 三问答得出，选题基本可用。

### 重难点 3：查新关键词构建

- **学生困难**：习惯用一句话整句检索，结果要么 0 条要么几万条。
- **突破**：用"对象 / 技术 / 应用"三组词拆解 + AND/OR 组合。**课堂示范一个完整例子**（讲义"海洋塑料垃圾"示例），让学生照葫芦画瓢。

### 重难点 4：模拟飞行手忙脚乱

- **学生困难**：四个通道同时控制，悬停极难。
- **突破**：不必追求完美悬停，达标标准设为"10 秒内漂移 <1 米"即可。**强调"飞控在替你做事"**——这正是传感器融合的价值，为下节传感器课埋伏笔。

## 常见问题（FAQ）

**Q1：学生选题"太大"怎么办？（如"我要做火星无人机"）**
> 引导**场景下切 + 尺度收窄**。不要否定热情，而是问："火星太远，我们能先做一个解决身边问题的版本吗？"例如火星无人机 → 极端环境无人机 → **高原/低温环境下无人机电池保温方案**。保留"航天感"但落到可验证原型。

**Q2：学生选题"太小"或像"手工课"怎么办？（如"做一个漂亮无人机模型"）**
> 引导**问题导向**。问："这个模型解决了什么真实问题？"把"做模型"改写为"用模型验证某个想法"。例如 → "不同螺旋桨桨距对悬停效率的影响研究"，模型只是验证手段，问题与原理才是核心。

**Q3：学生查不到相关文献怎么办？**
> 三个排查方向：① **关键词太窄**——尝试去掉一个限定词或换同义词（如"救援"→"应急→灾害响应"）；② **数据库选错**——航天类主题在知网用"航天/航空/遥感"分类检索；③ **方向确实太新**——这反而是好事，记录"查无相关"本身就是查新结论，但需在 Google Scholar 补检国际文献确认。

**Q4：组内对选题分歧大怎么办？**
> 两条路径：① **查新裁决**——让两组员各自用关键词检索，谁的选题"有研究空白可填"谁占优（用数据说话）；② **合并升级**——把两个方向融合（如"火情监测" + "北斗定位" = "基于北斗的森林火情监测无人机方案"）。若仍僵持，教师用评价量规判定，下节课前必须统一。

**Q5：学生想做"纯软件/编程"方向，与无人机硬件无关？**
> 允许。航天创意赛接受软件/算法类方案（如卫星轨道优化算法、遥感图像识别）。但要求其**科学原理与本课程知识相关**（航天器轨道、遥感、北斗等），不能做成普通的 App 开发项目。

### 任务 1（填空）
1. 对角线上的两个旋翼**同向**，相邻的两个旋翼**反向**，目的是**抵消反扭矩，防止机身自转**。
2. 要向前飞，需要**前**旋翼减速、**后**旋翼加速，使机头**前倾（下俯）**。
3. 实现偏航而不改变高度，需要**同向旋翼加速、反向旋翼减速**（如两个 CW 加速、两个 CCW 减速），此时总升力**不变**，但**反扭矩**不再平衡。
4. 都靠改变四个电机的**相对转速**实现，称为**差速控制**。

### 任务 2（连线）
- 飞控 → C（大脑）
- 电调 → B（神经）
- IMU → A（测姿态）
- 气压计 → D（测高）
- 磁力计 → E（定向）
- 光流传感器 → F（室内定位）

### 任务 3（法规判断）
1. ✘（0.3 kg 属"轻型"，微型 < 0.25 kg）
2. ✘（微型不强制实名登记，但鼓励）
3. ✓（禁飞区违规属违法）
4. ✓（小型需执照）
5. ✓（UOM 平台为官方登记渠道）

> ⚠️ 注：关于"微型是否强制实名登记"，条例原文为"除微型以外的无人驾驶航空器"应当登记。教师讲解时严格以此为准。学生易混淆，需强调。

## 差异化教学建议

- **基础薄弱学生/组**：选题可降低技术复杂度，重点保证"问题明确 + 原理正确"。模拟飞行允许只完成前 3 项（起飞/悬停/平移），不强求定向航线。查新数量可降至 3 条。
- **学有余力学生/组**：鼓励选题带**定量研究**色彩（如"X 参数对 Y 性能的影响"），查新要求扩展到 Google Scholar 并对比中外研究差异。模拟飞行挑战 FPV 穿越模式。
- **有编程基础的学生**：引导选题结合**无人机编程/自动航线**（对接"无人机编程技能挑战赛"），或做算法类航天创意方案。
- **偏文科兴趣学生**：引导做"航天法规/伦理/科普传播"类创意（如无人机法规科普短视频方案），但需保证科学原理准确。

## 赛事对接说明 🏆

本课是赛事对接的**关键训练节点**，对接三个层面：

1. **查新报告训练**（最重要）：期末作品包第 5 件 = 研究报告 + **查新报告**。查新报告是省级以上科创赛的**必备材料**，证明项目的"新颖性"。本课开始的查新训练，是为期末查新报告**积累素材**——现在认真记，期末直接成稿。
2. **无人机编程技能挑战赛认知**：讲法规时提及此赛道，让学生知道"航天创意赛"之外还有无人机实操赛道。对模拟飞行手感好的学生，可引导双线报名。
3. **选题锁定与团队稳定**：赛事要求团队稳定、选题一经提交原则不改。本课确认书即模拟赛事"作品申报"环节，让学生提前体验"立项即承诺"的严肃性。

> 🏆 **赛事节奏提醒**（重申）：3 月报名 → 6 月省赛 → 7–8 月国赛。本课（约学期中段）距省赛提交仅剩 **6–8 周**，选题必须本周定，否则后续无时间打磨。

## 附：选题打磨示范（教师备课用）

**原始意向（太空泛）**："用无人机做环境保护"

**三问打磨过程**：
1. 解决谁的问题？→ 巡护人员难以快速发现偏远林区火点
2. 用什么科学/技术？→ 无人机 + 红外热成像 + 北斗定位返航
3. 与现有方案不同？→ 现有方案多用单一可见光，本方案融合红外 + 北斗短报文回传（无信号区可用）

**立项后选题名称**：《基于红外热成像与北斗短报文的森林火情巡护无人机方案》

> 此案例体现"从空泛到具体、保留航天元素（北斗）、有明确创新点"的打磨路径，适合课堂示范。
