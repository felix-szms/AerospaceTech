## 💡 使用建议

1. **从预设场景入手**：依次点击「🛫 起飞」「🛩️ 巡航」「🪂 着陆」「⚠️ 失速」，直观对比不同飞行阶段参数差异。
2. **单变量探索**：保持其它参数不变，单独滑动一个滑块，观察升力变化曲线（提示：v 是平方关系，影响最大）。
3. **挑战失速边界**：把迎角从 5° 慢慢增大到 18°，观察流线动画 —— 你能看到上表面气流从整齐的层流变为紊乱的涡流。
4. **思考高度影响**：切换空气密度从 1.225 → 0.414，发现巡航所需速度必须更大才能维持同样升力（高空飞行为什么快？）。

> 💡 **关键直觉**：升力公式 $L = \tfrac{1}{2}\rho v^2 S C_L$ 中，速度 $v$ 是平方项，所以速度对升力的影响远大于其它参数 —— 这就是为什么飞机起飞、着陆都要关注速度，也是为什么失速如此危险。

## 🔬 拓展探究

| 探究问题 | 建议操作 |
| --- | --- |
| 飞行速度翻倍，升力变成几倍？ | 把 v 从 30 改到 60，观察 L 的变化 |
| 什么样的迎角能产生最大升力？ | 缓慢增大 α，记录每个角度的 C<sub>L</sub> |
| 高空巡航为什么更快？ | 同样 v=50、S=2 下对比 ρ=1.225 与 ρ=0.414 |
| 大飞机为什么机翼那么大？ | 对比 S=1 与 S=4 在相同速度下的升力 |

> ⚠️ **安全联想**：真实飞行中，飞行员永远关注**迎角指示器**而不是空速 —— 因为失速的本质是迎角过大，而非速度过低。
