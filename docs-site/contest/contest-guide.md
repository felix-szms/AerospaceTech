# 📖 赛事规则详解

> 全国青少年航天创新大赛 + 省市青少年科技创新大赛

## 一、全国青少年航天创新大赛（NYSC）

### 赛事概况

- **主办单位**：中国航天科技国际交流中心
- **赛事级别**：教育部白名单赛事
- **官网**：http://nysic.declare.htgjjl.com
- **承办周期**：每年一届

### 2025–2026 学年赛项设置

NYSC 设两大竞赛单元：

#### 单元一：航天科技创新单元（创意/探究/设计类 ⭐本课程重点）

| 赛项 | 性质 | 作品形式 |
|---|---|---|
| **航天创意赛（航天创意设计）** ⭐ | 创意方案+海报+视频+答辩 | 创意性+可行性导向 |
| **航天科技创新赛** ⭐ | 实物模型/编程作品+答辩 | 最接近工程实践 |
| 天文望远镜设计制作与探究赛 | 设计制作+探究报告 | 实物+报告 |

#### 单元二：太空探测竞技单元（动手/编程/机器人/无人机类）

| 赛项 | 性质 |
|---|---|
| 星球车机器人挑战赛 | 场地任务赛 |
| 火星家园机器人挑战赛 | 场地任务赛 |
| **无人机编程技能挑战赛** ⭐ | 编程+场地任务 |
| 无人机操作技能挑战赛 | 操作技能 |
| 球形无人机攻防对抗赛 | 对抗赛 |

### 重点赛项一：航天创意赛（本课程主对接）

| 维度 | 具体要求 |
|---|---|
| **面向组别** | 小学组 / 初中组 / **高中组**（含中专、中技、职高） |
| **参赛形式** | 团队 1–3 名学生 + 指导教师；每校每赛项最多 5 支队 |
| **比赛目标** | 激发航天兴趣，学习航天科学原理，掌握科学研究方法与工程设计方法 |
| **主题范围** | 载人航天、深空探测、北斗导航应用（每年可调整） |
| **提交作品** | ① 创意设计方案PDF(<5M) ② 展示海报(竖版≤900×1200mm) ③ 展示视频 ④ 方案实现（创意画/模型/其他） |
| **评审对象** | 创意方案 + 海报 + 方案实现 + 现场问辩 |
| **评分细则** | • 明确目的与问题（10 分）• 方案实现的过程（10 分）• 创意内容具象化呈现（10 分）• 思路清晰、分析正确、结论合理 |
| **答辩形式** | 全国总决赛为作品展演答辩 PPT 封闭评审 |

### 重点赛项二：航天科技创新赛

| 维度 | 具体要求 |
|---|---|
| **面向组别** | 小学组 / 初中组 / **高中组** |
| **作品形式** | **实物模型作品** 或 **编程作品** |
| **提交材料** | ① 创新设计方案文稿 ② 实物模型/编程作品 ③ 展示视频 ④ 展示海报 |
| **评审环节** | ① 航天科技知识考察（机答百分制）② 作品展演答辩（封闭评审） |
| **导向** | **解决问题为导向**，通过实验探究、技术创新、工程实践完成作品 |

### 时间节点

| 阶段 | 时间 |
|---|---|
| 大赛通知发布 | 每年 1 月 |
| 自助申报（官网报名） | 3 月起 |

| 全国总决赛（国赛） | 7–8 月 |

## 二、省市级青少年科技创新大赛（传统作品评审）

> 注：CASTIC 国赛已改革为现场挑战，但省市级仍保留传统作品评审模式。

### 评审标准："三自""三性"

**三自原则**（过程真实性，30%权重）

1. **自己选题**：选题须作者本人发现、提出、选择
2. **自己设计和研究**：创造性贡献须作者本人完成
3. **自己制作和撰写**：作者本人参与制作 + 撰写报告

**三性原则**（作品质量，70%权重）

| 性 | 含义 | 课程训练 |
|---|---|---|
| **科学性** | 原理正确、方法可靠 | 第 3、9、10 课理论 |
| **创新性** | 有新意（权重最高） | 第 6 课查新 + 第 16 课创意方法 |
| **实用性** | 能解决真实问题 | 第 1、6 课选题 |

### 申报材料

| 材料 | 要求 |
|---|---|
| 申报书 | 组委会标准模板 |
| 研究报告 | ≥2000 字，含设计思路、技术方案、测试数据 |
| **查新报告** | 研究开始前 + 申报前各做一次 |
| 查重报告 | 复制比符合各省阈值 |
| 诚信承诺书 | 申报者签署 |
| 项目展板 | 终评答辩用（通常 90cm×120cm 竖版）|
| 附件 | 实验记录、研究日志、实物、图纸 |

### CASTIC 国赛（2025 改革后）

国赛**不再评作品**，改为**现场科技挑战任务**，考察五大能力：

| 维度 | 含义 |
|---|---|
| **知识应用** | 综合运用科学知识解决实际问题 |
| **动手实践** | 现场操作、实验制作 |
| **创新思维** | 提出新颖方案与创意构想 |
| **批判精神** | 独立思考、质疑分析 |
| **团队协作** | 团队挑战中的沟通合作 |

> 前 10% 授予"中国科协主席奖"。选手须通过关联赛事（如航天创新大赛）获奖后遴选推荐。

## 三、能力培养对照表

| 能力点 | 对应赛事要求 | 课程培养 |
|---|---|---|
| ① 选题与问题发现 | CASTIC 三自；创意赛"明确目的" | 第 1、6 课 |
| ② 科学探究与查新 | 查新报告、研究报告 | 第 6 课查新训练 |
| ③ 工程设计与实现 | 科技创新赛"实物/编程" | 全程 EDP |
| ④ 创意表达（可视化） | 海报、创意画、方案 | 第 16-17 课 |
| ⑤ 科学写作 | 创意方案、研究报告 | 第 16-17 课 |
| ⑥ 知识储备 | 科技创新赛"知识考察" | 全程理论课 |
| ⑦ 答辩与展示 | 现场问辩 | 第 17-18 课 |
| ⑧ 团队协作 | CASTIC 五维之一 | 全程小组制 |
