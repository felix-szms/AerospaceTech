## 正文结构（按以下顺序撰写，建议正文 3000–6000 字）

### 一、研究背景与问题提出（约 800 字）

**1.1 研究背景**
- 本设计所处的航天领域现状（载人航天 / 深空探测 / 北斗应用 / 运载火箭 / 卫星应用……）。
- 相关的航天成就、技术发展、国家战略需求。
- 引用权威来源（国家航天局、论文、新闻报道），注明出处。

**1.2 发现的问题**
- 明确指出你要解决的具体问题：**谁**在**什么场景**下遇到了**什么困难**？
- 问题要具体，不要泛泛而谈。
- 最好有数据或案例支撑（如"据XX报告，……"）。

> 🏆 评分对应："明确目的与问题"（10 分）。评委会看你是否真的发现了一个真实、值得解决的问题。

### 二、查新与研究现状（约 600 字）

**2.1 文献与专利检索**
- 检索关键词：______
- 检索数据库：中国知网 / 国家知识产权局专利库 / Google Scholar
- 检索结果摘要（列出 3–5 条最相关的已有方案/研究）：

| 序号 | 来源（作者/标题/年份） | 核心内容 | 与本方案的异同 |
|---|---|---|---|
| 1 | | | |
| 2 | | | |
| 3 | | | |

**2.2 现有方案的不足**
- 对比分析已有方案的局限，说明你的方案为什么是必要的（"有什么新意"）。

### 三、科学原理（约 800 字）

- 你的方案依据哪些科学原理？（物理 / 航天 / 电子 / 材料等）
- 关键公式、定律、原理图。
- 这部分体现"科学性"——证明你的方案在科学上站得住。

> 示例：若设计北斗导盲手杖，需讲清卫星定位三角测量原理、北斗短报文通信原理、超声波测距原理。

### 四、创意设计方案（约 1200 字，核心章节）

**4.1 设计目标**
- 你的方案要实现什么功能？用 1–3 条明确的目标描述。

**4.2 总体方案**
- 系统总体架构（建议画一张**系统框图 / 结构图**）。
- 各组成部分及其功能。

**4.3 关键技术与创新点**
- 你的方案最"新"的地方在哪里？（技术新？场景新？方法新？集成新？）
- 这是评审"创意性"的核心，要重点论述。

**4.4 设计图纸**
- 结构图、原理图、三维模型截图等。
- 图要清晰、有标注。

### 五、技术路线与实现（约 800 字）

**5.1 实现步骤**
- 你的方案将如何实现？分步骤描述。
- 如果已制作原型，说明原型实现了哪些功能。

**5.2 关键器材 / 技术清单**

| 模块 | 器材/技术 | 型号/名称 | 作用 |
|---|---|---|---|
| | | | |

**5.3 预期难点与解决思路**
- 实现过程中可能遇到的技术难点，及你的应对方案。

### 六、可行性分析（约 600 字）

**6.1 科学可行性**：原理是否成立？
**6.2 技术可行性**：现有技术能否实现？核心模块是否有成熟方案？
**6.3 经济可行性**：成本估算（原型级别）。
**6.4 原型验证情况**（如有）：你做了什么测试，结果如何？

> 🏆 评分对应"方案实现的过程"（10 分）。评委会看你是否真的动手做过、验证过。

### 七、应用前景与展望（约 400 字）

- 你的方案未来能用在哪些场景？有什么推广价值？
- 如果未来技术进步，方案还能怎么升级？

## ⚠️ 注意事项

1. **方案文稿不得与申报书装订在一起**（赛事要求）。
2. **查新报告须单独提供**（见查新报告模板）。
3. 所有图表建议自制或标注来源，避免版权问题。
4. 团队成员须为实际参与者，指导教师署名须真实。
5. 赛事规则每年微调，提交前务必核对当年最新版"创意设计方案模板"。
