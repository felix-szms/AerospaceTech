# 查新报告模板

> 🏆 对接：全国青少年航天创新大赛 + 省市青少年科技创新大赛
> 查新报告是赛事必备材料，须在**研究开始前**和**申报前**各做一次。
> 建议使用中国知网（CNKI）、国家知识产权局专利检索系统、Google Scholar 进行检索。

## ✅ 查新报告自检清单

- [ ] 检索了至少 2 个数据库。
- [ ] 关键词组合合理（核心词+同义词）。
- [ ] 列出至少 5 条相关结果。

- [ ] 查新人和指导教师签字。
- [ ] 与研究报告/方案文稿分开装订。

## 💡 教师指导要点

1. **立题查新**（第 6 课布置）：学生在确定选题时做，目的是确认选题有价值、不重复。此时尚无完整结果，重点是关键词检索和现状了解。
2. **申报查新**（第 16–17 课完成）：作品基本完成后做，目的是正式出具查新报告提交赛事。需要更系统地检索和对比。
3. **常见问题**：学生容易"检索不到就写没查到"。要引导：检索不到完全相同的结果，正说明有新意——但要如实记录检索过程，让评委看到你是认真查过的。
4. **专业查新机构**：部分省赛要求发明类作品提供专业部门（如知网查新中心）出具的查新报告，建议提前了解本省要求。
