# 课程总览

> 《空天科技探秘》面向高一年级，18 课时 × 45 分钟，航空 8 + 航天 8 + 综合创意设计 2。

## 课程定位

| 项目 | 说明 |
|---|---|
| **课程名称** | 空天科技探秘 |
| **适用对象** | 高一年级（15–16 岁） |
| **课时安排** | 18 课时 × 45 分钟，每周 1 课时 |
| **模块划分** | 航空模块（8）+ 航天模块（8）+ 综合创意设计（2） |
| **核心教学法** | PBL（顶层）+ EDP（工程循环）+ 5E（单课节奏） |
| **理论:实操** | 约 1:2（理论 6 课时，实操与项目 12 课时） |

## 课程目标

完成本课程后，学生能够：

1. **理解** 航空与航天的基本原理（空气动力学、轨道力学、火箭推进、卫星技术）。
2. **掌握** 工程设计过程（EDP），独立完成"提问→构思→计划→制作→测试→改进"的完整循环。
3. **动手制作** 飞行器/火箭/卫星模型，运用传感器与编程进行数据采集与任务控制。
4. **完成** 一套航天创意设计作品包（方案文稿+模型+海报+视频+研究报告+答辩），具备参赛能力。
5. **发展** 科学探究、工程思维、团队协作、创意表达与答辩展示等综合素养。

## 教学法骨架

```
┌─────────────────────────────────────────────┐
│   PBL（项目式学习）—— 顶层架构，学期长项目    │
│   ┌─────────────────────────────────────┐   │
│   │  EDP（工程设计过程）—— 每个项目的骨架  │   │
│   │  Ask → Imagine → Plan → Create      │   │
│   │  → Test → Improve（循环）            │   │
│   │  ┌───────────────────────────────┐  │   │
│   │  │ 5E（每节课的节奏）             │  │   │
│   │  │ Engage → Explore → Explain     │  │   │
│   │  │ → Elaborate → Evaluate         │  │   │
│   │  └───────────────────────────────┘  │   │
│   └─────────────────────────────────────┘   │
└─────────────────────────────────────────────┘
```

## 贯穿全课程的 PBL 项目

**项目名称**：《______ 的航天创意设计》（学生自选主题）

**作品包六件套**——直接对接赛事申报：

| 编号 | 交付物 | 对接 |
|---|---|---|
| 1 | 创意设计方案文稿（PDF） | 航天创意赛核心 |
| 2 | 实物模型 / 概念原型 | 科技创新赛 |
| 3 | 展示海报（竖版 ≤900×1200mm） | 现场展示 |
| 4 | 展示视频（3–5 分钟） | 材料评审 |
| 5 | 研究报告（≥2000 字）+ 查新报告 | 省市科创赛 |
| 6 | 答辩 PPT + 研究日志 | 现场答辩 |

**PBL 里程碑节点**：

| 课时 | 里程碑 |
|---|---|
| 第 1 课 | 发布项目任务，组建团队 |
| 第 6 课 | 🏆 中期选题确认 + 查新 |
| 第 14 课 | 🏆 原型/方案定稿 |
| 第 16-17 课 | 集中打磨作品包 |
| 第 18 课 | 🏆 全班答辩会（模拟国赛）|

## 评估方式

| 评估维度 | 占比 | 方式 |
|---|---|---|
| 过程性记录（研究日志/EDP记录单） | 30% | 每课记录单 + 日志抽查 |
| 实操作品（航模/火箭/CanSat/无人机任务） | 20% | 作品质量 + 测试数据 |
| PBL 作品包（方案+模型+海报+视频） | 30% | 第 16-17 课交付 |
| 答辩展示 | 20% | 第 18 课模拟国赛 |

## 赛事节奏建议

| 时段 | 课程进度 | 赛事动作 |
|---|---|---|
| 9–12 月（秋季） | 第 1-10 课 | 选题、查新、知识储备、原型制作 |
| 1–2 月（寒假） | 第 11-15 课 | 完善作品、研究报告、海报、视频 |
| 3 月 | 第 16-18 课 | 集中打磨 + 答辩；**NYSC 报名** |
| 3–6 月 | 课后延伸 | **省赛/选拔赛** |
| 7–8 月 | 课后延伸 | **国赛**；优秀作品申报 CASTIC 遴选 |
