# 实践活动：手掷滑翔机制作与试飞配平

> 第 5 课实践环节（约 35 分钟）
> 教学法：EDP 之 Test → Improve（聚焦配平迭代）

---

## 活动目标

1. **动手组装**一架手掷滑翔机，识别机翼上反角、水平尾翼、垂直尾翼、升降舵等关键部件。
2. 通过**至少 3 轮试飞—诊断—配平**迭代，让飞机达到稳定平飞。
3. 用《试飞配平记录表》完整记录工程过程，养成"调整—效果"对应的工程日志习惯（直接对接赛事工程实践记录要求）。
4. 在留空挑战中体会"配平好坏 = 性能高低"。

## 活动流程

### Phase 1 · 组装（10 分钟）✅

1. **清点零件**：对照套件说明书，确认机翼（含左右两半）、机身、水平尾翼、垂直尾翼、机头配重、连接件齐全。
2. **按说明组装**，特别注意以下三个"关键部位"，组装完成后**用记号笔/贴纸标注**：
   - 🔵 **上反角位置**：左右机翼拼接处是否形成 V 形？（从正前方平视应能看到）
   - 🟢 **水平尾翼**：是否与机身水平面平行？是否端正不歪？
   - 🔴 **垂直尾翼**：是否与机身对称面重合、不左偏不右偏？
3. **初装重心检查**：用两根手指顶在机翼根部下方，找平衡点，看重心是否落在机翼前缘向后约 **1/3 弦长**处。偏后→机头加配重；偏前→暂时记录，试飞后再说。
4. 在机翼或机身写上**组别**，避免试飞时拿混。

> 🎤 教师巡视重点：① 上反角是否到位（最常见错误是机翼装平了）；② 垂尾是否歪斜；③ 502 胶水使用安全。

### Phase 2 · 基线试飞（5 分钟）✅

到指定试飞区，**先做一次"诊断试飞"**，建立基线：

- **投掷要领**：手握机身重心附近，机头略微朝上（约 5°–10°），**水平向前、平稳推送**——不是用力"甩"，而是"放"出去。投掷高度约与肩齐平。
- **观察重点**：飞机离手后**最先出现什么问题**？是抬头？低头？侧倾？偏头？平稳？
- 把第 1 次飞行姿态记入记录表"第 0 轮（基线）"行，作为对比基准。

> 💡 投掷手法本身会影响观察。建议每组固定 1 名"投掷手"以保证一致性，避免"飞机没病，是扔的问题"。

### Phase 3 · 诊断与配平迭代（15 分钟）✅

这是本课的**核心环节**。按"**飞一次 → 诊断 → 调整 → 再飞**"循环，**至少完成 3 轮**。

**诊断对照表**（详见讲义与教师参考）：

| 病症 | 现象 | 主要原因 | 调整措施 |
|---|---|---|---|
| 波飞（porpoising）| 海豚式上下震荡 | 重心太后 / 升降舵上偏过多 | 机头加配重（橡皮泥/回形针）；或升降舵微微下偏 |
| 栽葱（diving）| 出手即低头俯冲 | 重心太前 / 升降舵下偏过多 | 机头减重；或升降舵微微上偏 |
| 偏航（yawing）| 机头侧滑、打转 | 垂尾歪斜 / 左右翼不对称 | 校正垂尾；检查左右翼上反角是否对称 |
| 翻滚（rolling）| 绕纵轴打转 | 上反角不足 / 左右翼不对称 | 轻扳机翼增大上反角；校正机翼对称 |
| 失速（stall）| 机头猛抬后掉高度 | 迎角过大 / 投掷过猛上仰 | 升降舵微下偏；投掷角度降低 |

**每轮必须记录**：飞行现象 → 诊断原因 → 采取什么调整 → 调整后效果（改善/无变化/恶化）。

> ⚠️ **每次只调一个变量**！同时改重心又改升降舵，你将无法判断是哪个起作用——这是工程实验的基本纪律，也是赛事评审看重的"严谨性"。

### Phase 4 · 留空挑战（5 分钟）🏆

配平基本满意后，进入"留空比拼"：

- 每组 **2 次正式投掷**，取**最长留空时间**为成绩。
- 同时记录**飞行距离**（从投掷线到落点）。
- 全班横向比较，留空时间最长的小组获"本课最佳试飞员"称号。

> 💡 进阶挑战：尝试"轻掷"和"略带仰角掷"两种手法，看哪种留空更长——这背后是能量管理（势能↔动能）的学问。

## 讨论问题（如时间允许）

1. 你们组改了哪个变量效果最明显？为什么？
2. 如果让你把这架滑翔机改成"飞得最远"而不是"飞得最久"，你会怎么调？（提示：考虑下滑角度与速度）
3. 上反角越大的飞机越稳，那为什么战斗机上反角几乎为零？
4. 真实飞机（如客机）的配平由飞行员还是自动系统完成？这给了你什么启发？

## ⚠️ 安全提示（务必遵守）

1. **场地选择**：在**操场草坪、体育馆、空旷大厅**等开阔场地试飞；**严禁**在教室、走廊、楼道投掷。
2. **远离危险区**：试飞区必须**远离人群、道路、停车场、电线、变压器、水域**；机头虽钝，但飞行速度下仍可能伤眼。
3. **轮流规则**：每组试飞时，**一次只允许一组投掷**，其余同学站在指定"观察区"，**严禁迎面接飞机**。
4. **工具安全**：美工刀使用时刀刃**朝外、远离手指**；502 胶水**切勿入眼、粘手**，用后盖紧；KT 板边角锋利，注意防划伤。
5. **回收纪律**：飞机落地后由本组成员自行取回，**不得到马路/围墙外捡拾**；卡在树上/电线上**不要攀爬**，报告教师处理。
6. **天气**：户外试飞选择**无雨、风力 ≤ 3 级**的天气；风大时改室内或调整投掷方向（逆风起飞）。

> 🎤 教师课前须踩点试飞场地，划定"投掷线"与"观察区"，并准备急救包（创可贴、洗眼液）。
