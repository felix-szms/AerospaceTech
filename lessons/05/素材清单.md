# 第 5 课 · 素材清单

## 一、核心器材：手掷滑翔机套件

本课成败的关键是**套件选型**。建议教师**课前自己先买 1–2 款试装**，选定后再批量采购。

### 套件类型对比

| 类型 | 价格 | 优点 | 缺点 | 适用场景 |
|---|---|---|---|---|
| **KT 板冲压套件**（推荐入门）| 10–25 元/套 | 便宜、耐摔、易用剪刀/胶带修改、容错高 | 精度低、上反角难固定、飞行性能一般 | 大班首课、初学者 |
| **轻木激光切割套件**（推荐进阶）| 30–50 元/套 | 精度高、飞行性能好、更像真飞机、可教蒙皮工艺 | 易断、需 502 胶、对动手要求高 | 小班、学有余力组 |
| **EPS 泡沫成型整机**（备选）| 15–30 元/套 | 出厂即装好、免组装、即拆即飞 | 无法体验组装过程、配平调整空间小 | 时间紧、只想试飞 |

### 采购参考（教师备课，非指定品牌）

> 以下为常见品类，**非指定型号**。教师可在淘宝/京东/拼多多搜索关键词比价，选口碑好的店铺。

| 搜索关键词 | 说明 |
|---|---|
| "手掷滑翔机 套件 KT板" | 入门首选，常带预切件 + 配重 + 说明书 |
| "轻木 滑翔机 模型 套材" | 进阶用，含轻木板/梁/翼肋 + 蒙皮纸 |
| "手抛飞机 滑翔机 整机 EPS" | 免组装备选，EPP/EPS 一体成型 |
| "航空模型 手掷机 教学" | 部分店铺有教师装批发价 |
| "回旋镖/手掷机 教学套件" | 部分品牌有配套教案 |

**采购建议**：
- **数量**：班级人数 ÷ 2.5（每组 2–3 人）+ 备用 2–3 套（防损坏）。
- **到货检查**：① 零件是否齐全（机翼、机身、水平尾翼、垂直尾翼、配重）；② KT 板有无折痕；③ 轻木有无断裂。
- **预算参考**：40 人班级（约 16 组），KT 板方案约 **300–500 元**，轻木方案约 **600–900 元**。

> 💡 **建议**：首次开课统一用一种类型（便于指导），熟练后可 KT 板 + 轻木混搭，让进阶组挑战轻木。

### 推荐选型原则

1. **优先选带说明书和标注关键部位（上反角、重心点）的套件**——大幅降低教学难度。
2. **机翼分左右两半、需拼接形成上反角的**优于一体机翼——学生能亲手"做出"上反角，理解更深。
3. **带可调升降舵（尾翼后缘可折）的**优于固定尾翼——否则无法练习配平核心环节。
4. **避免电动/遥控滑翔机**——本课要的是"纯气动配平"，电动会掩盖气动问题。

## 三、场地要求

### 试飞场地标准

| 要求 | 说明 |
|---|---|
| **开阔度** | 长度 ≥ 15m、宽度 ≥ 8m（手掷滑翔机典型飞行距离 5–15m）|
| **无障碍** | 场地内及延伸方向**无**人群、道路、停车场、电线、变压器、水域、玻璃 |
| **地面** | **首选草坪**（飞机落地不易坏）；次选塑胶跑道、木地板（体育馆）；**避免**水泥地（易摔坏）、瓷砖（太滑）|
| **室内备选** | 体育馆、空旷大厅、长走廊（≥20m）；室内无风干扰，配平更纯粹 |
| **分隔** | 能划定"投掷线"与"观察区"（粉笔/胶带标记）|

### 场地踩点清单（教师课前完成）
- [ ] 丈量场地尺寸是否够（≥15m×8m）
- [ ] 确认投掷方向延伸区无道路/电线/水域
- [ ] 确认可划定投掷线与观察区
- [ ] 确认地面类型（草坪最佳）
- [ ] 确认室内备选方案（防天气变化）
- [ ] 准备粉笔/胶带做标记

## 注意事项 ⚠️

1. **课前必须自己组装 1 架并配平成功**——这是教师指导的底气，也是示范用的实物。
2. **502 胶水、美工刀由教师管控**，使用时当面指导，避免伤害。
3. **备用套件留 2–3 套**，轻木尤其易损，损坏会直接影响小组进度。
4. **天气预案**：户外课查当日风力与降水，风力 ≥4 级或下雨立即转室内方案。
5. **器材保管**：本课组装的滑翔机后续课程（第 6 课复盘、期末展示）仍需使用，提醒学生带回妥善保管，标注组别防混。
6. **⚠️ 预算敏感**：若学校经费有限，KT 板方案 + 回形针配重（替代橡皮泥）+ 美纹纸胶带（替代双面胶）可把每组成本压到 **10 元以内**。
