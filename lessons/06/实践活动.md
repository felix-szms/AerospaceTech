## 活动目标

## 活动一：模拟飞行基础操控（15 分钟）

### 飞行前准备

1. 打开模拟飞行软件，选择**四旋翼（Quadcopter）**机型。
2. 选择**无风、开阔场地**场景（如草地、操场）。
3. 确认遥控器通道映射：
   - **左摇杆**：上下 = 油门（Throttle，升降）；左右 = 偏航（Yaw，自转）
   - **右摇杆**：上下 = 俯仰（Pitch，前后）；左右 = 横滚（Roll，左右）

> 💡 这是"美国手（Mode 2）"布局，全球最通用，大疆等商用机也采用此布局。

### 五项基础操控练习

按下表逐项练习，每项先由教师示范，再由学生操作。每完成一项打 ✓。

| 序号 | 任务 | 操作要点 | 完成标准 | ✓ |
|---|---|---|---|---|
| **1** | **起飞** | 缓慢推油门至 50% 左右，无人机离地 1–2 米 | 稳定离地不侧倾 | |
| **2** | **悬停** | 微调油门保持高度，修正俯仰/横滚保持位置 | 在 2 米高度停留 **≥10 秒**，漂移 < 1 米 | |
| **3** | **平移** | 悬停基础上轻推俯仰/横滚，控制方向移动 | 向前 5 米 → 停 → 退回原位 | |
| **4** | **定向飞行** | 先偏航转 90°，再沿新方向平移前进 | 完成一个"正方形航线"（前进→转向×4）| |
| **5** | **降落** | 移至起降点正上方，缓慢减油门，触地即停 | 轻柔着陆，无反弹、无翻机 | |

每位操控者填写：

| 操控者姓名 | 悬停时长（秒）| 着陆评分（1–5）| 摔机次数 | 体会/困难 |
|---|---|---|---|---|
| | | | | |
| | | | | |

> 💡 **反思提问**：在模拟器里，你最难控制的是哪一项？为什么？（多数学生答"悬停"——因为它要求四个通道同时微调，这正是飞控在真实无人机上每秒帮你做几百次的事。）

### 挑战任务（💡 如时间允许）

完成基础五项后，尝试：
- **绕标飞行**：围绕一个标志物做圆形航线（考验俯仰 + 偏航协调）。
- **穿越窄门**：从两个障碍物之间穿过（考验精确定位）。

✅ **完成后**：每组推选操控最好的一位，在全班面前演示"正方形航线"。

## 活动二：PBL 选题确认工作坊（15 分钟）🏆【关键里程碑】

> 本环节是整门课 PBL 的**分水岭**：在此之前的选题都是"意向"，今天之后就是**立项**。请认真对待。

### 工作坊流程

1. **组内回顾**（3 分钟）：重温第 1 课提交的《团队选题意向表》，讨论过去 5 周是否有方向调整。
2. **关键词初拟**（2 分钟）：把你们想解决的问题浓缩成 3–5 个关键词（对象 + 技术 + 应用）。
3. **填写确认书**（7 分钟）：完成《选题确认书》全部字段（见下）。
4. **教师审核**（3 分钟）：教师巡回，对每组选题给出"通过 / 修改 / 重新选题"意见。

## 查新实操指导

### 第一步：构建检索式

把确认书第五栏的"对象 / 技术 / 应用"三组关键词，用 `AND`（且）、`OR`（或）、`()`（分组）组合。

**例**：选题"基于北斗短报文的户外救援手环"

```
检索式：(北斗 OR Beidou) AND 短报文 AND (救援 OR 应急 OR 求救)
```

### 第二步：在知网检索

1. 打开 https://www.cnki.net/（校内通常有权限）
2. 选择"高级检索"
3. 输入检索式，来源类别可选"期刊 / 硕博 / 会议"
4. 按**被引量**或**相关度**排序，挑出最相关的 5–10 篇

### 第三步：在专利库检索

1. 打开国家知识产权局专利检索 https://pss-system.cponline.cnipa.gov.cn/
2. 在"名称/摘要"栏输入关键词组合
3. 重点关注近 5 年的**发明专利**（实用新型含金量较低）

> 🏆 **查新报告的意义**：期末作品包第 5 件要求"研究报告 + 查新报告"。你现在记录的每一篇文献，都是查新报告"已有方案对比"和"本方案创新点"的素材。**现在认真记，期末省一半力气**。

> 💡 **怎么判断"相关"？** 一篇文献如果同时提到你的关键词中的 2 个以上，且研究方向接近，就算相关。完全无关的不要凑数。

## 讨论问题（如时间允许）

1. 模拟飞行中，**悬停**为什么比直线飞行更难？（提示：四个通道耦合、无稳态）
2. 如果让你设计一架用于**森林火情监测**的无人机，你会选哪种类型（微型/轻型/小型）？为什么？需要哪些传感器？（提示：考虑续航、载荷、法规、GPS + 红外）
3. 在查新中，如果你发现自己的创意**已经有人做过**，该怎么办？（提示：找差异点 → 细分场景 → 改进方法 → 这正是创新的起点）

## ⚠️ 安全提示

**模拟飞行环节**：
- 模拟器虽无实物风险，但仍需**控制操作节奏**，避免猛推摇杆——养成轻柔操作习惯，将来真机飞行更安全。
- 长时间盯屏幕注意用眼，每 15 分钟远眺一次。

**查新环节**：
- 知网等数据库账号仅限校内学习使用，**不得批量下载、转售文献**，违反可能被封号。
- 引用文献必须**标注出处**，抄袭/不标注将影响查新报告与研究报告的真实性评分。

**真机提醒（本课不涉及，但需知晓）**：
- 真实无人机锂电池充电必须有人值守，使用平衡充电器。
- 任何真机飞行前，确认所在区域**不在禁飞区**，并已完成**实名登记**。
