|---|---|---|---|
| A8-3 | | | |

| C6-5 | | | |

判断题（✓ / ✘）：

- ( ) 字母越靠后，发动机总冲量越大。
- ( ) A 级发动机的总冲量是 B 级的 2 倍。
- ( ) 编码最后一位数字代表推力持续时间。
- ( ) 延迟时间越长，开伞时火箭一定飞得越高。

## 课后任务

### 必做

1. **优化你的 OpenRocket 设计**：尝试把最高点精确控制在 **95–105 m**，记录你的最终参数与仿真截图，存入团队 PBL 文件夹（赛事方案素材）。

   我的最终设计：
   - 发动机：__________  最高点：______ m  稳定性裕度：____ 口径
   - 关键调整：________________________________

2. **写一段（100–150 字）的"仿真依据"**：用你的仿真数据为火箭设计创意赛方案写一段科学依据说明，包括目标高度、稳定性裕度、选发动机的理由。这段话可以直接进方案文稿。

3. **整理组装检查清单**：把未完成项补完，下节课带到发射课。

### 选做 💡

1. **研究不同发动机的影响**：在同一套设计上，分别换成 A8-3、B4-4、C6-5，仿真对比最高点、最大速度、稳定性，写一份小报告。

2. **探究稳定性裕度**：把尾翼尺寸调小到一半，仿真会发生什么？再把鼻锥加重 10 g，仿真又如何？记录"稳定性裕度"的变化。

3. **查阅 SpaceX / 蓝箭航天的可回收火箭**：他们的回收方式（着陆腿 vs 降落伞）和你的模型火箭回收有何异同？
