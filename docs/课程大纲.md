# 《空天科技探秘》课程大纲

> 面向高一年级 · 18 课时 × 45 分钟 · 航空 8 + 航天 8 + 综合创意设计 2
> 教学法骨架：PBL（学期长项目）+ EDP（工程设计循环）+ 5E（单课节奏）

---

## 一、课程定位

| 项目 | 说明 |
|---|---|
| 课程名称 | 空天科技探秘 |
| 适用对象 | 高一年级（15–16 岁） |
| 课时安排 | 18 课时 × 45 分钟，建议每周 1 课时 |
| 模块划分 | 航空模块（8 课时）+ 航天模块（8 课时）+ 综合创意设计（2 课时） |
| 核心教学法 | PBL（顶层）+ EDP（工程循环）+ 5E（单课节奏） |
| 理论:实操 | 约 1:2（理论 6 课时，实操与项目 12 课时） |

## 二、课程目标

完成本课程后，学生能够：

1. **理解**航空与航天的基本原理（空气动力学、轨道力学、火箭推进、卫星技术）。
2. **掌握**工程设计过程（EDP），能独立完成"提问→构思→计划→制作→测试→改进"的完整循环。
3. **动手制作**飞行器/火箭/卫星模型，并运用传感器与编程进行数据采集与任务控制。
4. **完成**一套航天创意设计作品包（方案文稿+模型+海报+视频+研究报告+答辩），具备参加全国青少年航天创新大赛的能力。
5. **发展**科学探究、工程思维、团队协作、创意表达与答辩展示等综合素养。

## 五、赛事对接

### 主对接赛事

**全国青少年航天创新大赛 · 航天创意赛（高中组）**
- 主题：载人航天 / 深空探测 / 北斗应用等
- 形式：团队 1–3 人，创意方案+海报+视频+模型+答辩
- 评审：创意性 + 可行性
- 时间：3 月报名 → 3-6 月省赛 → 7-8 月国赛

### 辅助对接赛事

- **航天科技创新赛**（实物模型/编程作品）
- **无人机编程技能挑战赛**
- **省市级青少年科技创新大赛**（工程学/计算机类，传统作品评审）
