# 文档规范

## 一、文件命名

| 文件 | 文件名 |
|---|---|
| 学生讲义 | `讲义.md` |
| 实践活动 | `实践活动.md` |
| 学生任务单 | `学生任务单.md` |
| 教师参考 | `教师参考.md` |
| 素材清单 | `素材清单.md` |

## 二、讲义.md 结构（学生用）

```markdown
# 第 X 课 · 标题

> 模块：航空/航天/综合 ｜ 课时：45 分钟 ｜ 类型：理论/实操/项目

## 学习目标
3-5 条，用"能够…"开头，可观测可评估。

## 核心知识
### 1. 知识点一
（正文，含原理、公式、图示说明）
…
## 本课小结
要点回顾。
## 延伸阅读
推荐链接/视频/书目。
```

## 三、实践活动.md 结构（学生用）

```markdown
# 实践活动：XXX

## 活动目标
## 器材清单（表格）
## 活动步骤
### 步骤 1：…
### 步骤 2：…
## 数据记录表
## 讨论问题
⚠️ 安全提示
```

## 四、教师参考.md 结构（教师用）

```markdown
# 第 X 课 · 教师参考

## 课时概览
## 时间分配表（45 分钟细分）
## 教学设计（5E 或 EDP 对应）
## 重难点与突破策略
## 常见问题（FAQ）
## 参考答案（对应学生任务单）
## 差异化教学建议
## 赛事对接说明 🏆
```

## 五、格式约定

- ✅ 学生活动 ｜ 🎤 教师讲授 ｜ 🏆 赛事对接 ｜ ⚠️ 安全提示 ｜ 💡 高阶拓展
- 公式用 LaTeX 行内 `$...$` 或块级 `$$...$$`
- 外部资源注明来源和链接
- 代码块标注语言
