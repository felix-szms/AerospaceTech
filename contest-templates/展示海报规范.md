# 展示海报规范与设计指南

> 🏆 对接：全国青少年航天创新大赛 + 省市青少年科技创新大赛
> 提交格式：JPG，竖版，尺寸 ≤ 900mm × 1200mm（即宽度≤900mm，高度≤1200mm），单文件 < 5MB。

---

## 一、海报基本规范

| 项目 | 要求 |
|---|---|
| 版式 | **竖版**（纵向） |
| 尺寸 | 宽 ≤ 900mm × 高 ≤ 1200mm（约 3:4） |
| 分辨率 | ≥ 150 DPI（打印清晰） |
| 格式 | JPG（提交）；建议同时保存源文件（AI/PSD/Canva） |
| 文件大小 | < 5MB |
| 配色 | 建议深色太空背景（深蓝/黑），白色或亮色文字，科技感 |

---

## 二、海报内容结构（官方要求 + 优秀实践）

赛事要求海报含以下板块，建议按此布局：

```
┌─────────────────────────────────┐
│         标题区（顶部）            │
│  作品名称 + 团队成员 + 学校 + 组别  │
├─────────────────────────────────┤
│                                 │
│   ① 设计意图 / 问题背景          │
│   （解决什么问题，为什么）         │
│                                 │
├────────────────┬────────────────┤
│                │                │
│  ② 总体方案    │  ③ 设计图纸    │
│  系统框图      │  结构图/三维图  │
│  关键创意      │  关键参数       │
│                │                │
├────────────────┴────────────────┤
│                                 │
│   ④ 细节展示                     │
│   关键模块特写 / 实物照片         │
│   测试数据图表                   │
│                                 │
├─────────────────────────────────┤
│   ⑤ 总结与展望                   │
│   创新点 / 应用前景               │
├─────────────────────────────────┤
│   团队简介 + 指导教师 + 致谢      │
└─────────────────────────────────┘
```

### 各板块要点

**① 标题区**
- 作品名称醒目（最大字号）。
- 副标题一句话点明方案核心。
- 团队成员姓名、学校、组别、指导教师。

**② 设计意图 / 问题背景**
- 用 50–100 字说清"为谁解决什么问题"。
- 配一张场景图（如用户痛点场景）。

**③ 总体方案与设计图纸**
- **系统框图**（最重要）：清楚展示各模块如何协同。
- 三维结构图或工程图。
- 关键参数表（尺寸、重量、性能指标）。

**④ 细节展示**
- 关键模块特写照片或渲染图。
- 测试数据用**图表**呈现（柱状图/折线图，不要纯文字）。
- 实物照片（如有原型）。

**⑤ 总结与展望**
- 3 条核心创新点（bullet 形式，简洁有力）。
- 应用场景。
- 未来改进方向。

**底部**
- 团队简介（一句话）。
- 指导教师、致谢（高校合作/校外专家）。

## 三、设计原则

### 1. 视觉层次（评委 3 秒抓重点）
- **标题最大**（评委远处第一眼）。
- **图表 > 文字**（评委更愿意看图）。
- 文字段落尽量短，多用 bullet。
- 关键数据/创新点用**高亮色**或**图标**强调。

### 2. 图表优先
- 系统框图、结构图、数据图占海报面积的 **50% 以上**。
- 所有图表有**编号和标题**（"图 1 系统总体架构"）。
- 避免：大段纯文字、密密麻麻的表格。

### 3. 配色与风格
- 航天主题推荐深色背景（深蓝 #0a1929 / 黑色）+ 亮色文字（白/金/青）。
- 主色不超过 3 种，避免花哨。
- 可用航天元素点缀（星球、轨道线、火箭图标），但不要喧宾夺主。

### 4. 留白
- 板块之间留呼吸空间，不要塞满。
- 评委在嘈杂的决赛现场扫海报，**留白比信息量更重要**。

## 四、推荐设计工具

| 工具 | 类型 | 适合 | 备注 |
|---|---|---|---|
| **Canva**（可画）| 在线，免费 | 新手首选，有科研海报模板 | canva.com |
| **Figma** | 在线，免费 | 协作设计，矢量图 | 适合团队协作 |
| **Adobe Illustrator / Photoshop** | 桌面，付费 | 专业输出 | 学校机房或有授权 |
| **PowerPoint / Keynote** | 桌面，易得 | 简单快捷 | 设自定义页面 900×1200mm |
| **Affinity Designer** | 桌面，一次性付费 | AI 替代 | 性价比高 |

> 💡 建议学生用 Canva（免费、模板多、易上手），或 PowerPoint（人人会用）。专业需求用 AI。

## 五、常见问题与改进

| 问题 | 改进 |
|---|---|
| 文字太多，像论文 | 删减文字，转为图表/bullet |
| 没有系统框图 | 必加，这是评委理解方案的入口 |
| 图模糊 | 提高分辨率，用矢量图或高清渲染 |
| 配色混乱 | 主色≤3种，统一风格 |
| 看不出创新点 | 创新点单独成块，高亮强调 |
| 排版拥挤 | 增加留白，减少内容（精简是能力）|
| 没有实物照片 | 如有原型，务必放（增强可信度）|

## 六、参考案例（供学生观摩）

> 教师可准备 2–3 份历年优秀海报（脱敏）供学生观摩，重点分析其：

> - 创新点如何突出
> - 视觉层次如何引导阅读

## 💡 教师指导要点

1. **第 16-17 课集中制作**：海报是作品包的"门面"，安排充足时间。
2. **先内容后设计**：让学生先确定各板块文字内容，再做视觉设计，避免"先排版后凑字"。
3. **同伴互评**：让学生互相看海报，模拟"3 秒抓重点"——同伴 3 秒没看懂的部分就要改。
4. **打印预览**：务必按实际尺寸打印预览，检查清晰度和比例。
