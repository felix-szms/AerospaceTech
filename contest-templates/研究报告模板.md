# 研究报告模板

> 🏆 对接：省市级青少年科技创新大赛（工程学 / 计算机科学 / 物理类）
> 要求：中学生项目研究报告一般 **≥ 2000 字**。本文本结构同样适用于航天科技创新赛的"创新设计方案"文稿。

## 摘要（200–300 字）

用一段话概括：**为什么做（背景与问题）→ 做了什么（方案与创意）→ 怎么做的（方法）→ 做出了什么（结果）→ 有什么意义（结论）**。

> 摘要是评委首先看的内容，务必精炼有力。

**关键词**：______；______；______（3–5 个，用分号分隔）

---

## 正文

### 一、研究背景（300–500 字）

- 本项目所处的领域背景、国家需求、社会意义。
- 你发现的具体问题是什么？（数据/案例支撑）

### 二、研究目的与意义（150–300 字）

- 你想解决什么问题？
- 解决它有什么意义？（科学价值 / 社会价值 / 应用价值）

### 三、研究方法（400–800 字）

**3.1 总体思路**
- 用什么方法解决问题？（实验探究 / 工程设计 / 技术开发 / 调查研究）

**3.2 技术路线**
- 画一张**技术路线流程图**，展示从问题到方案的完整路径。

**3.3 具体方法**
- 实验/设计/制作的详细步骤。
- 用到的器材、软件、技术。
- 数据采集与处理方法。

> 🏆 这部分对应评审"三自"中的"自己设计和研究"——要体现你的创造性贡献。

### 四、设计与实现（800–1500 字，核心章节）

**4.1 方案设计**
- 总体架构（系统框图）。
- 关键模块设计。
- 创新点说明（你的方案与已有方案的区别）。

**4.2 制作与实现**
- 实物/原型的制作过程。
- 关键技术难点的解决。
- 配实物照片、制作过程照片。

**4.3 关键参数 / 设计计算**
- 涉及的计算（如尺寸、参数选择、性能估算）。
- 用公式和数值说明设计依据。

### 五、测试与分析（400–800 字）

**5.1 测试方案**
- 测什么？怎么测？测试条件？

**5.2 测试数据**

| 测试项目 | 测试条件 | 结果 | 目标值 | 是否达标 |
|---|---|---|---|---|
| | | | | |

**5.3 数据分析**
- 数据说明什么？方案是否有效？
- 与预期对比，分析偏差原因。

### 六、结论与展望（200–400 字）

**6.1 结论**
- 本项目完成了什么？验证了什么？解决了什么问题？

**6.2 不足与改进方向**
- 坦诚说明方案的不足（评委会欣赏诚实的自我反思）。
- 未来如何改进。

**6.3 应用展望**
- 方案的应用场景和推广价值。

### 七、参考文献

格式：
```
[序号] 作者. 标题. 期刊/出版社, 年份, 卷(期): 页码.
```

至少列出 5 条，中外文均可。

### 八、附录（可选）

- 实物照片 / 设计图纸 / 程序代码 / 原始数据表
- 研究日志摘要

## 💡 评审"三自""三性"对照（教师/学生参考）

省市级科创赛传统评审标准：

**"三自"原则（过程真实性）**
- ✅ **自己选题**：选题须作者本人发现、提出 → 对应"研究背景"
- ✅ **自己设计和研究**：创造性贡献须作者本人完成 → 对应"研究方法""设计与实现"
- ✅ **自己制作和撰写**：作者本人参与制作 + 本人撰写报告 → 全文须自己写

**"三性"原则（作品质量）**
- ✅ **科学性**：原理正确、方法可靠 → 对应"科学原理""研究方法"
- ✅ **创新性**：有新意（通常权重最高）→ 对应"创新点说明""查新结论"
- ✅ **实用性**：能解决真实问题 → 对应"应用展望"

> 撰写报告时，对照这六条逐项确保到位。
